# example-chat-web-backbone
Backbone implementation of a chat using example-chat-backend project.

## How to use
    npm install example-chat-web-backbone
    npm start example-chat-web-backbone

## Technologies used
* Backbone
* ReactJS
* Socket.io
* ES6 (Babel)
* Gulp
* Browserify
* Watchify
* Stylus
* Autoprefixer
