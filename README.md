# example-chat-web-backbone
Backbone implementation of a chat using example-chat-backend project.

## How to use
    npm install example-chat-web-backbone
    node node_modules/example-chat-web-backbone/server/index.js

## Technologies used
* Backbone
* ReactJS
* Socket.io
* ES6 (Babel)
* Gulp
* Browserify
* Watchify
* Stylus
* Autoprefixer
